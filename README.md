widenbot-gif
----

An (work-in-progress) gif plugin for widenbot.

# Commands

```
!gif
```
# Installation

1. Add dependency to your bot project:

```
npm install --save widenbot-gif
```

2. Enable in config `plugins`:

```
module.exports = {
    plugins : {
        // ...
        gif: {}
    }
}
```
